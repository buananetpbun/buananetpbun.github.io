SNTP Client or NTP Network Time Protocol - MikroTik Script RouterOS

NTP is the Network Time Protocol, a protocol used to perform time concerts in a network.

# How to setup SNTP Client in Mikrotik
/system ntp client
set enabled=yes primary-ntp=111.92.175.248 secondary-ntp=130.54.208.201 server-dns-names=asia.pool.ntp.org
