Block Twitter with "Layer 7" or "Content" or "TLS" - MikroTik Script RouterOS

How to Block Twitter with "Layer 7" or "Content" or "TLS"

Block Twitter With "Layer-7"

/ip firewall layer7-protocol
add name=Twitter regexp="^.+(twitter.com|twimg.com).*\$"
/ip firewall filter
add action=drop chain=forward layer7-protocol=Twitter

Block Twitter With "Content"

/ip firewall filter
add action=drop chain=forward content="twitter.com" 
add action=drop chain=forward content=".twitter."
add action=drop chain=forward content="twimg.com" 
add action=drop chain=forward content=".twimg."

Block Twitter With "TLS"

/ip firewall filter
add action=drop chain=forward protocol=tcp tls-host="twitter.com"
add action=drop chain=forward protocol=tcp tls-host="*.twitter.*"
add action=drop chain=forward protocol=tcp tls-host="twimg.com"
add action=drop chain=forward protocol=tcp tls-host="*.twimg.*"
